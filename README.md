# YooMooc RN

哟慕课 APP 是一个第三方的优慕课客户端

## 🎞背景

无论是电脑版还是手机版的优慕课客户端用起来似乎都不是那么令人满意，电脑版在课程页面只能看到标题不能看到详情，手机版从详情页退出后会回到最顶端。于是就有了做一个第三方客户端的想法，也当是 React Native 的一次练手了
